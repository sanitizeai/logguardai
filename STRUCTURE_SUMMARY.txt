SCALABLE FOR FUTURE VERSIONS:

v0.3 Release Process:
  1. mkdir docs/versions/v0.3
  2. Copy & update v0.2 files
  3. Update symlink: rm docs/latest && ln -s v0.3 docs/latest
  4. Update CHANGELOG.md
  5. Done! Users automatically see v0.3 as latest

Result:
  docs/versions/v0.1/ (accessible, archived)
  docs/versions/v0.2/ (accessible, stable)
  docs/versions/v0.3/ (latest, current)
  docs/latest/ → v0.3/ (users always see latest)
